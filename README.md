## Quick start (Installation and Usage)
Install [Node.js&reg;](https://nodejs.org)

Install dependencies:
 ```
 npm install
 ```

Start the server:
```
npm start
```

Use _LoIDE_ in a browser at:
```
http://localhost:8084
```
